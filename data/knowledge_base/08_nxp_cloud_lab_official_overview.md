资料来源：NXP AIoT Cloud / Cloud Lab 官网首页、站点地图和公开前端页面。官网标题将 Cloud Lab 定位为面向嵌入式设计的一站式云上开发平台。站点导航包含硬件、AI Hub、软件、系统方案、应用范例、大学生竞赛、技术支持等入口，适合作为本项目 RAG 知识库的领域资料来源。

结构化摘要：

| 主题 | 官方页面体现 | 可用于客服回答的问题 |
| --- | --- | --- |
| Cloud Lab 总览 | 首页与导航 | NXP Cloud Lab 是什么、能体验哪些内容、为什么适合作为嵌入式/边缘 AI 学习入口 |
| 硬件资源 | 产品页与开发板详情页 | i.MX、i.MX RT、Layerscape、MCX 等系列如何通过云实验室了解或预约体验 |
| AI Hub | `/ai-solution` 相关页面 | NXP 如何支持边缘 AI 模型集成、模型部署、模型转换和 AI 方案体验 |
| 软件工具 | `/software` 与 DDR/Voice Plugins 入口 | DDR 参数配置、语音插件、多媒体框架工具有什么用途 |
| 系统方案 | `/solution` 与方案详情页 | EVSE 等方案如何结合 NXP 开发板和云端实验环境展示 |
| 应用范例 | `/use-case` 与 use-case-show 页面 | 用户指南、产品功能体验、培训和支持资料如何帮助开发者上手 |
| 技术支持 | `/support` | 用户遇到实验、开发板、软件工具问题时如何获取支持 |

站点地图中公开列出了大量可作为后续知识库扩展对象的页面，包括 i.MX 9、i.MX 8、i.MX 6、i.MX RT、Layerscape、MCX、合作伙伴板卡、开发板详情、应用范例和文章。当前本地知识库优先整理 AI Hub、边缘 AI 示例和软件工具，因为这些内容与项目要求的 RAG、Agent 工作流和本地模型问答最相关。
