与 i.MX 应用范例的区别：

- i.MX 应用范例更偏 Linux MPU、显示、多媒体、内核驱动和边缘 AI。
- MCU 应用范例更偏 MCX 系列微控制器上的轻量视觉或传感应用。
- 合作伙伴应用范例用于承载第三方板卡或生态方案，当前接口没有可见条目。
